Previsor de Demanda

Este repositório contém um aplicativo Streamlit para previsão de demanda com médias móveis, suavização exponencial e regressão linear.

Importante

O GitHub Pages só hospeda sites estáticos. Como este app é escrito em Python/Streamlit, ele não pode rodar diretamente via GitHub Pages.

Como executar localmente:
  1. Clone o repositório.
  2. Instale as dependências: pip install streamlit plotly pandas numpy scikit-learn.
  3. Execute: streamlit run documento.py.
